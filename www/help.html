About This App

This is a typing app developed keeping in mind, the children with autism and other special needs. The app has three levels for kids with different levels of functioning. The Beginner and Advanced level present a virtual keyboard. The key to be pressed is highlighted to help with scanning and focussing on the right key in the keyboard layout. In the beginner level, only one key is highlighted at a time. In the Advanced level, all the keys in the word are highlighted to prompt the child make a selection. The Expert level present the actual iOS keyboard. Only the correct keystroke is captured and incorrect responses lead to audio prompt. All the level present auditory prompts (audio of next alphabet to be pressed)and visual prompts (highlighting next alphabet).

List Of Available Words

The app provides with preloaded Dolch Word lists - Pre-primer, Primer, 1st Grade, 2nd Grade, 3rd Grade and Nouns. The Dolch word lists are non editable. They can be viewed in View-Edit Word List settings. In addition to preloaded Dolch lists, Custom word list is available at View-Edit Word List/Custom Word List. The Custom word list is editable. New words be be stored in device local storage.

Adding/Deleting Words

Users can add new words in the custom word list at View-Edit Word List/Custom Word List, by clicking to the Add button. Any new word may only include alphabets, space, comma and fullstop. Words can be deleted from Custom Word List by clicking the delete button and selecting the words to be deleted.

Selecting Word List

Users can select the word list to use by going to Select Word List setting. More than one list can be selected. If no list is selected, default selection (Dolch Word List - preprimer) will be made. If the Custom word list is empty, it will not be visible for selection.

Setting Display Order Of Words

Three display orders are available - Default(as appearing in the list), Alphabetical and Random. Order of words can be set from Display Order setting.

Recording Custom Words

For each word, the audio for the word is played followed by audio prompt of individual letters. The audio for Dolch Words is preloaded. Users can record the custom words that they add in the Custom Word list. Audio can be recorded after adding new words using Record Custom Words Settings. For best result to avoid overlapping of sounds, length of the audio should be 2 sec or less.

Keyboard Types and Sizes

The Beginner and Advanced levels uses a virtual keyboard with key highlight feature for visual prompt and scanning. There are two sizes for virtual keyboard - The standard size similar to the ios keyboard and the big size keyboard for children having motor coordination or visual challenges. The virtual keyboard size can be selected from Select Virtual Keyboard Setting. The Expert level uses the standard iOS keyboard.

Incorrect Response

In the Advanced level, it may be possible that the typed word is spelled incorrectly. In case of incorrect response, the same word is displayed for next attempt, after two attempts the correct spelling is displayed and the next word is selected.
